FAQ

The most asked questions are answered here.

How to download?

There is a download tutorial on our download page.

Is it safe?

The client is 100% safe. How do you know? Well it is really simple, the code is publicly available to everyone to see and change on Github. This means if we put any virus in the client you can see it and also remove it.

Who are you?

We are just 2 guys who like coding and thought this would be a fun project to make. So I hope you are enjoying the project.

How to open the Click Gui?

The Click Gui is binded to RIGHT_SHIFT and can be changed in the Controls settings.

Who are the people who made the client originally?

MatHax Client is an Meteor Client fork.
Check them out here.

Does this Client work with Optifine?

Optifine currently doesnt work with MatHax Client and causes many issues with it.

Can i replace Optifine with something other?

MatHax Client works with well with Sodium, Phosphor and Lithium.

Why did you do this?

We thought it would be cool to make or remake an Utility Client and create a Website for it. We are also learning from this.

How can i make my own client?

If you really want go to the Meteor Client Github here download all the code and start coding in your preferred IDE.
